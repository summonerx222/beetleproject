# Музей любимых жуков

Готовый статический сайт для портфолио: галерея любимых насекомых, поиск, фильтры, модальное окно с фактами и декоративные жуки на фоне при скролле.

## Что внутри

- `index.html` — структура сайта
- `styles.css` — стили, анимации, адаптив
- `data.js` — контент карточек
- `script.js` — логика галереи, фильтров, модалки и фона
- `assets/` — SVG-иллюстрации насекомых

## Как запустить локально

Достаточно открыть `index.html` в браузере.

## Как загрузить на GitHub и Cloudflare

1. Создай новый репозиторий на GitHub.
2. Загрузи в него все файлы проекта целиком вместе с папкой `assets`.
3. В Cloudflare открой **Workers & Pages**.
4. Нажми **Create** → **Import a repository**.
5. Подключи GitHub и выбери этот репозиторий.
6. Для статического проекта build command можно оставить пустой.
7. Output directory укажи как `.`
8. Запусти деплой.

## Что можно быстро поменять

- Тексты в шапке — в `index.html`
- Список насекомых — в `data.js`
- Цвета и стили — в `styles.css`
- Поведение карточек и анимации — в `script.js`
